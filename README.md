# Serviços
- Grafana
- Prometheus
- MYSQL
- MYSQL-Exporter
- InfluxDB
- Telegraf
- Node-Exporter
- Cadvisor
- BlackpExporter
# Dependências
- Docker
- Git
# Build
Para executar todos os serviços é apenas seguir os seguintes comandos:

``` git clone https://github.com/robertasolimandonofreo/grafana-docker.git ```
``` cd grafana-docker ```
``` docker-compose up --build```
