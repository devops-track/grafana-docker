# Serviços em execução no docker-compose
- Grafana
- Prometheus
- MYSQL
- MYSQL-Exporter
- InfluxDB
- Telegraf
- Node-Exporter
- Cadvisor
- Black-Exporter
# Dependências
- Docker
- Git
# Build
Para executar todos os serviços é apenas seguir os seguintes comandos:

``` git clone htps://github.com/robertasolimandonofreo/grafana-docker.git ```

``` cd grafana-docker ```

``` docker-compose up --build ```
